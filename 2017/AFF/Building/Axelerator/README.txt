Repeatability package for Axelerator 
Dario cattaruzza, April 2017.

Please extract the package into the desired folder and follow the instructions in install.txt to install dependencies and build the tool.
The script for the building benchmark is found in the benchmarks folder. It will output the results both to the console and the files Building.ld and Building.mpi for unsound and sound evaluations respectively. The output is a vector in the direction of x25 with its corresponding maximum.
